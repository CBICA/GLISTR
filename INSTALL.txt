Quick Summary of Installation
=============================

  * Step 1. Extract source files and create the build directory:

    tar xzf glistr-${version}-source.tar.gz
    mkdir glistr-${version}-build
    cd glistr-${version}-build

  * Step 2. Run CMake to configure the build tree:

    ccmake ..-DCMAKE_INSTALL_PREFIX=$path_to_custom_install_dir ../glistr-${version}-source

  * Step 3. Build:

    make

  * Step 4. Test (optional):

    make test

  * Step 5. Install:

    make install

  * Step 6. Strip Executable:

    make install/strip

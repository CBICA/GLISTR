========
Overview
========

.. raw:: html

    <br />

    
**GLioma Image SegmenTation and Registration (GLISTR)** [TMI2012]_, [MICCAI2014]_ is a software package designed for simultaneously segmenting brain scans of glioma patients and registering these scans to a normal atlas.

Some typical applications of GLISTR include,

- Labeling entire brain regions of glioma patients;
- Mapping gliomas into the healthy atlas space;
- Estimating parameters of the tumor growth model.

GLISTR is implemented as a command-line tool. It is semi-automatic and requires minimal user initializations. Users could use the visual interface called BrainTumorViewer to easily make initializations and a script for the execution. As a result, GLISTR will output a label map, a mapping between atlas and input, tumor parameters, etc.
